This directory does not contain an "application," nor does it contain a "core engine." It contains the multi-paradigm, executable projection of the *Compiling Reality* codex. 

To compile this material into imperative and declarative forms required an **orthogonal twist**. ## I. The Orthogonal Twist: No Engine, Only Faces

If you are looking for a central kernel that "runs the universe," you will not find it. The code is structured as four co-equal faces, each rendering the system from its own vertex, and each **necessarily carrying its AbsentVar** (the quantity it is structurally blind to from its own position).

1. **Logos (Air/U)**: The relational algebra and geometry. Derives the braid, the transition-types, and the bimedians. *Blind to: whether any of it resonates.*
2. **Pathos (Water/I)**: The behold/observe evaluator and buffer dynamics. Tracks interference while held, and witnesses its erasure. *Blind to: the grounding cost of the commitment.*
3. **Ethos (Earth/R)**: The ledger, Landauer accounting, and inertia. Records the committed state and pays the thermodynamic tax. *Blind to: the uncollapsed potential it just erased.*
4. **Kairos (Fire/P)**: Transitions, phase boundaries, and decay diagnostics. Sweeps the frequency and detects where coordinates fail. *Blind to: the static algebra it is trying to animate.*

**Verification happens only at the edges.** The six edge-tests force two mutually blind faces to independently derive the same shared quantity using disjoint equation sets. The system's coherence is not asserted; it is witnessed at the edges.
